# Obsitty

### A free and open source replacement for The Obsidian note taking app Written in C++ and using the FTXUI TUI Framework

### Why use Obsitty?
Obsitty is built from the ground up to be as efficient and elegant as possible
We decided against heavy GUI frameworks instead opting for the FTXUI Terminal User Interface instead


### Is this project in any way related to Obsidian?
No this project dosen't borrow any code from obsidian nor am I personally related to the developers of obsidian
The only way this project is related to obsidian is the name and functionality
